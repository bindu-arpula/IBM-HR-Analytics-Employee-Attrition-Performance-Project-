# IBM-HR-Analytics-Employee-Attrition-Performance-Project-
Built an interactive HR Analytics dashboard to identify employee attrition trends, workforce demographics, and business performance indicators using Microsoft Excel.

KPIs:
Total Employees
Attrition Rate
Active Employees
Average Age
Average Income
Average Years at Company

Tools Used:
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Validation

HR Analytics Dashboard using Excel:
Built a business intelligence dashboard from raw HR data.
Performed exploratory data analysis using Pivot Tables and Excel formulas.
Created dynamic KPI cards and interactive visualizations.
Presented employee attrition insights across departments, age groups, education fields, and overtime status.
Improved decision-making through data-driven workforce analysis.
